## psql

ssh 서버에서 아래의 명령어를 입력하면 postgreSQL로 접근가능하다.

```bash
su postgres
psql
```

psql 도움말 - `\?`

sql 도움말 - `\h`

특정 sql 쿼리에 대한 도움말 - `\h select`

database 전체 목록 확인 - `\l`

테이블 목록 확인 - `\d or \dt`
테이블 상세 정보 확인 - `\d+`

schema 확인 - `\ds`

다른 database 사용하기 - `\c Database 이름`

psql 종료 - `\q`
