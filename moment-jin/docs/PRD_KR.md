## 2. 개요

### 2.1 제품 소개

Moment-Jin은 크롬의 기본 새 탭 페이지를 생산성 중심의 대시보드로 교체하는 확장 프로그램입니다. 시간 관리, 작업 추적, 근무 시간 기록, 알림 관리를 아름답고 커스터마이징 가능한 인터페이스에서 통합 제공합니다.

### 2.2 타겟 사용자

- **주요**: 통합된 생산성 도구가 필요한 지식 근로자, 개발자, 재택근무자
- **부차**: 여러 작업과 일정을 관리하는 학생과 프리랜서

### 2.3 핵심 가치 제안

- 외부 의존성 없는 올인원 생산성 대시보드
- 여러 브라우저 탭 간 자동 날짜 동기화
- Chrome Storage API 기반 오프라인 우선 설계
- 다국어 지원 (한국어, 영어, 일본어, 중국어)
- 개인화된 워크플로우를 위한 커스터마이징 가능한 패널 표시
- Unsplash의 아름다운 동적 배경

## 3. 제품 목표 및 성공 지표

### 3.1 목표

1. 컨텍스트 전환 없이 원활한 일일 생산성 추적 제공
2. 여러 브라우저 탭 간 실시간 동기화 지원
3. Chrome Storage API를 통한 데이터 영속성 및 신뢰성 보장
4. 포괄적인 i18n으로 국제 사용자 지원

### 3.2 성공 지표

- 사용자 참여도: 설치 기반의 80% 이상 일일 활성 사용
- 데이터 신뢰성: 99.9% 성공적인 스토리지 작업
- 성능: 100ms 미만의 초기 페이지 로드
- 탭 간 동기화 지연: 1초 미만

## 4. 기능 요구사항

### 4.1 핵심 레이아웃 및 내비게이션

#### REQ-001: 반응형 레이아웃 시스템

**우선순위**: P0 (Critical)  
**설명**: 애플리케이션은 가로 및 세로 화면 방향 모두에 적응형 레이아웃을 제공해야 합니다.

**완료 조건**:

- 가로 화면: 사이드 패널 (왼쪽 즐겨찾기, 오른쪽 할일/근무기록)
- 세로 화면: 하단 패널 (알림, 할일, 근무기록 스택)
- 브라우저 창 회전/크기 조정 시 부드러운 전환
- 모든 표준 화면 해상도(1024px ~ 4K)에서 레이아웃 깨짐 없음

**의존성**: 없음

---

#### REQ-002: 다국어 지원 (i18n)

**우선순위**: P0 (Critical)  
**설명**: 브라우저 언어 자동 감지와 함께 4개 언어 지원.

**완료 조건**:

- 지원 언어: 한국어(ko), 영어(en), 일본어(ja), 중국어(zh)
- 첫 실행 시 브라우저 언어 자동 감지
- 모든 UI 텍스트 일관되게 번역
- 날짜/시간 형식이 로케일 규칙 준수
- 수동 언어 전환 가능

**의존성**: 없음

### 4.2 메인 대시보드 영역

**우선순위**: P0 (Critical)  
**설명**: HH:MM 형식으로 현재 시간을 표시하고 자동 업데이트.

**완료 조건**:

- 로케일 기반 12/24시간 형식
- 큰 가독성 좋은 폰트 크기 (4rem)
- 페이지 로드 시 부드러운 페이드인 애니메이션

**의존성**: 없음

#### REQ-004: 개인화된 인사말

**우선순위**: P1 (High)  
**설명**: 시간대별 인사말을 선택적 사용자 이름과 함께 표시.

**완료 조건**:

- 시간 기반 인사말: 아침(05:00-12:00), 오후(12:00-17:00), 저녁(17:00-22:00), 밤(22:00-05:00)
- 형식: "{인사말}, {사용자명}" 또는 이름 미설정 시 "{인사말}"만
- 지원하는 모든 언어에 대해 현지화된 인사말 메시지
- 설정에서 이름 변경 시 자동 업데이트

**의존성**: REQ-002, REQ-021

**설명**: 자주 방문하는 웹사이트를 위한 빠른 접근 패널.

- 접을 수 있는 패널 (확장/축소)
- 설정에서 선택적 패널 표시

#### REQ-008: 즐겨찾기 패널 상태

**우선순위**: P1 (High)  
**설명**: 패널은 부드러운 전환과 함께 열림/접힘 상태를 지원해야 합니다.

**완료 조건**:

- 접힘 상태: 세로 표시만

- 세션 간 상태 유지
- 부드러운 CSS 전환 (300ms)

**의존성**: REQ-007

### 4.4 할일 패널

#### REQ-009: 작업 관리

**우선순위**: P0 (Critical)  
**설명**: 날짜별로 구성된 일일 작업 생성, 관리 및 추적.

**완료 조건**:

- Enter 키로 작업 추가
- 날짜별로 작업 그룹화 (YYYY-MM-DD)
- 현재 날짜에 "오늘" 배지 표시
- 남은/전체 개수 표시
- 체크/체크 해제로 완료 표시
- 확인 후 작업 삭제
- 날짜순 정렬 (최신순)
- 완료된 작업은 시각적으로 구별 (취소선, 불투명도 감소)

**의존성**: REQ-012

---

#### REQ-010: 날짜 기반 작업 필터링

**우선순위**: P1 (High)  
**설명**: 시각적 표시와 함께 선택한 날짜로 작업 필터링.

**완료 조건**:

- 날짜 헤더 클릭하여 선택
- 선택된 날짜 강조 표시
- 선택된 날짜에 새 작업 추가 (선택 없으면 오늘)
- 선택 취소 버튼
- 선택된 날짜로 자동 스크롤
- 날짜 형식: "MM/DD (요일)" (현지화된 요일명)

**의존성**: REQ-009

**설명**: 복잡함을 줄이기 위해 완료된 작업 표시/숨김 옵션.

**완료 조건**:

- 눈 아이콘 토글 버튼 (👁️/👁️‍🗨️)

- 필터가 즉시 적용

- 오늘 항목 강조

**의존성**: REQ-013

### 4.6 알림 패널

#### REQ-016: 알림 생성

**우선순위**: P1 (High)  
**설명**: 여러 리마인더 간격으로 시간 알림 생성.

**완료 조건**:

- 필수 필드: 제목, 목표 날짜 및 시간
- 선택적 필드: 설명
- 여러 리마인더 타이밍:
  - 정시 (정확한 순간)
  - 5분 전
  - 10분 전
  - 30분 전
  - 1시간 전
  - 1일 전
- 알림 활성화/비활성화
- 기존 알림 편집
- 확인 후 삭제

**의존성**: REQ-017

---

#### REQ-017: 알림 표시

**우선순위**: P1 (High)  
**설명**: 시간 기반 스타일링으로 다가오는 알림 표시.

**완료 조건**:

- 지난 알림은 회색으로 표시
- 비활성화된 알림은 불투명도 감소
- 메인 헤더에 다음 알림 표시
- 리마인더 시간에 크롬 알림
- 날짜/시간순 정렬 (가장 빠른 것 먼저)

**의존성**: REQ-018

### 4.7 패널 관리

#### REQ-020: 드래그 앤 드롭 순서 변경

**우선순위**: P1 (High)  
**설명**: 드래그 앤 드롭 인터랙션으로 즐겨찾기 순서 변경.

- 변경 사항 자동 저장
- 데스크톱과 터치 기기 모두 작동
- 실수 방지 (임계값: 5px)

**의존성**: REQ-007

#### REQ-021: 설정 모달

**우선순위**: P1 (High)  
**설명**: 모든 앱 설정을 위한 중앙 집중식 구성.

**완료 조건**:

- 톱니바퀴 아이콘(좌하단)으로 열기
- 섹션:
  - 사용자 이름 입력
  - 패널 표시 토글 (즐겨찾기, 할일, 근무, 알림, 핵심목표)
  - 데이터 내보내기/가져오기

## 9. 제약사항 및 가정

### 9.1 제약사항

- Chrome 확장 프로그램만 (v1.0에서는 다른 브라우저 없음)
- Chrome Storage 할당량 제한 (sync 100KB, local 10MB)
- Manifest V3를 위해 Chrome 120+ 필요
- 영문 문서만 제공

### 9.2 가정

- 사용자가 Chrome 브라우저 설치
- 사용자가 초기 로드를 위한 인터넷 연결
- 사용자가 Chrome 확장 프로그램 설치에 익숙
- 사용자가 영어 UI에 익숙 (또는 지원 언어의 원어민)
