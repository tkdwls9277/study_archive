# update

기존 DATA를 변경하는 쿼리문

## 문법

```sql
UPDATE 변경하는 테이블

SET 변경하는 컬럼 = 값

WHERE 변경할 조건
```

-   변경하는 테이블 : DATA를 변경할 테이블을 명시합니다.

    필수로 명시해야 하는 부분입니다.

-   변경하는 컬럼 : 변경할 컬럼들을 나열합니다.

    필수로 명시해야 하는 부분입니다.

-   값 :  변경할 컬럼에 대입되는 값을 명시합니다.

    필수로 명시해야 하는 부분입니다.

-   변경할 조건 : 해당 테이블에서 변경되야 할 대상의 조건을 명시합니다.

    조건이 필요하지 않은 경우에는 생략 가능한 구절입니다만,

    DATA의 변경이 일어나는 만큼 신중해야 할 구문입니다.
